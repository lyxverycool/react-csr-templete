# 运用技术栈

`react v16.8 + webpack v4.x+ react-router-dom v5.0.0 + babel v7 + antd`

# 环境目录

`env`

# 安装

`npm install`

# 运行(开发模式)

`npm run start || npm run dev`  

`env==development react-hot-loader热更新`

# 构建

`npm run build` 

`env==production 打包在dist 目录下`
